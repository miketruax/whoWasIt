Who Was It?

A simple app to help you remember the actor's name in a couple movies. Soon there will be implementation to find a movie based off actor's in it.

Technologies Used:

  • Angular 2 (via Angular CLI for ease of scaffolding)
  • Bootstrap CSS/JavaScript for easier design.
  • OMDBApi for search results
  • jQuery for easier AJAX calls
