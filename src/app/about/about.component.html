A simple app to help you remember the actor's name in a couple movies. As soon as I can find a good open API option, there will be implementation to find a movie based off actor's in it. Currently limited by the information from OMDBApi in that not all actors are populated and IMDB does not allow scraping, otherwise that would be the best option.

Technologies Used:

  • Angular 2 (via Angular CLI for ease of scaffolding)
  • Bootstrap CSS/JavaScript for easier design.
  • OMDBApi for search results
  • Ngrx and Rxjs for Stores and Observables
